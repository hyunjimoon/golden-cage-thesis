## Q2: Empirical Gap — Sell × Deliver

**현재 상황**:
- P(S∧D|B) = P(S|B) × P(D|B) 프레임워크 제시
- dG/dA > 0, dA/dF < 0 는 실증됨

**질문**: 이 gap이 논문의 약점인가?

| Option | Description |
