> "최초 진입자가 시장을 지배한다"

| Netscape (브라우저) | 사라짐 | Chrome |

## 🧩 Puzzle: First Mover ≠ First Winner

> "왜 최초 진입자가 갇히는가?"

**Motional의 상황**:
```
세계 최초 AV 스타트업
$4B 투자 (Hyundai + Aptiv JV)
2024년: 40% 해고
결과: M ≈ 0 (갇힘)
```

**공식으로 설명**:
```
학습 조건: μ(1−μ) < ε(τ+1)

First Mover의 저주:
1. 최초 → 구체적 선택 (τ↑)
2. τ↑ → 비낙관자 배제
3. μ → 1 (Echo Chamber)
4. μ(1−μ) → 0 (학습 불가)
5. 갇힘
```

**일반화**:
- Biotech의 "기술 먼저" 접근
- Deep Tech의 긴 개발 주기
- 모든 High-Commitment 산업

## 🔍 Lightsaber: 3가지 탈출로

### 탈출로 1: 기대관리 (Aurora)

```
전략: "우리는 바뀔 수 있다" 선언
메커니즘: 구체적이어도 피벗 허용 확보
적용: 투자자/파트너에게 명시적 flexibility 확보
```

| Motional | Aurora |
|:---|:---|
| 단일 OEM (Hyundai) | Multi-OEM |
| 피벗 = 배신 | 피벗 = 학습 |

### 탈출로 2: 전략적 모호성 (Tesla)

```
전략: 선택적 High-V 유지
메커니즘: 어디를 구체화하고 어디를 모호하게 둘지 선택
```

| 차원 | Tesla | Motional |
|:---|:---|:---|
| 기술 (HOW) | Camera-only (구체적) | LiDAR (구체적) |
| 시점 (WHEN) | "곧" (모호) | 2026 (구체적) |
| 정체성 (WHAT) | Car/Energy/AI? (모호) | L4 Solution (구체적) |

**Tesla = Strategic Ambiguity의 교과서**

### 탈출로 3: Provisional Commitment

```
4가지 원칙:
┌─────────────────────────────────────────────────────────────────────────┐
│  1. Temporary: "지금은 이렇게, 나중에 바뀔 수 있음"                     │
│  2. Mutual Confirmatory: "양측이 합의한 조건부 약속"                    │
│  3. Explicit Uncertainty: "불확실성을 숨기지 않음"                      │
│  4. Immediate Action: "결정을 미루지 않고 행동"                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**적용 예시**:
| 상황 | 기존 | Provisional |
|:---|:---|:---|
| OEM 파트너 | "현대 전용" | "현대 우선, 다른 OEM 열림" |
| 기술 선택 | "LiDAR 확정" | "LiDAR 시작, 데이터 기반 재평가" |
| 시장 | "B2B only" | "B2B 우선, B2C 옵션 유지" |

## 일반화: First Mover's Curse 패턴

| 산업 | First Mover | Curse | 탈출한 자 |

| AV | Motional | Corporate Lock-in | Aurora, Tesla |

**탈출한 자의 공통점**:
- 어딘가에 모호성 유지 (Tesla)
- 또는 명시적 피벗 허용 (Aurora)
- 또는 플랫폼화 (Moderna)
